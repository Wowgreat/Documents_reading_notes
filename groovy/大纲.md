##### 动态添加属性/方法

```groovy
class Person{
    
}

Person.metaClass.name_ = null //动态添加属性，注意 name是个关键字，在groovy闭包中指向当前类名

Person.metaClass.nameUpperCase = {return name_.toUpperCase()}
def person = new Person(name_: 'neo', age: 26)
println(person.nameUpperCase())
// neo: 36
// NEO
```
