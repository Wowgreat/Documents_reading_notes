#### 3.6.1 Condition Codes

- CPU保留一组单bit的 *Condition Code* 寄存器，用来描述最近的数学或逻辑运算的属性。

  | code | express |
  | ---- | ------- |
  | CF   | 进位符  |
  | ZF   | 零符    |
  | SF   | 符号符  |
  | OF   | 溢出符  |
