#### 异常处理

#### 异常种类

| Class 类型     | Cause原因          | Async/Sync 异步/同步 | Return Behavior 返回行为 |
| -------------- | ------------------ | -------------------- | ------------------------ |
| 中断 Interrupt | 来自I/O设备的信号  | 异步                 |                          |
| 陷阱 trap      | 故意的异常         | 同步                 |                          |
| 故障 fault     | 潜在的可恢复的错误 | 同步                 |                          |
| 终止 abort     | 不可恢复的错误     | 同步                 |                          |
